# id
# Id app Built with  react native

## Installation

To install, go to the root directory and run:

### `npm install`

## Emulate On Android
To emulate on android device go to the root directory and run:

### `npm run debug`

## Emulate On iOS
To emulate on iOS device go to the root directory and run:


### `run debug-ios`
